# Privacy Policy for Online Now

**Last Updated**: January 24, 2026

## Introduction

Online Now ("the App") is committed to protecting your privacy. This Privacy Policy explains how the App handles information.

## Our Privacy Commitment

**We do not collect, store, transmit, or share any personal data from users of the App.**

## Data Collection

**The App collects ZERO data.**

### What We Don't Collect

We do not collect:
- Personal information (name, email, phone, etc.)
- Location data
- Device information
- Usage analytics
- Crash reports
- Advertising identifiers
- Any other personal or non-personal data

## Local Data Storage

### Connection History

The App stores connection check history locally on your device using iOS UserDefaults. This data includes:
- Timestamp of each check
- Connection type (Wi-Fi, Cellular, or None)
- Online/Offline status
- Speed measurement results (when available)

**Important**: This data:
- Never leaves your device
- Is not transmitted to any server
- Is not accessible to us or any third party
- Can be deleted by you at any time using the "Clear History" button

## Network Requests

The App makes only two types of network requests, both solely for functionality:

### 1. Internet Reachability Check
- **Purpose**: Verify actual internet connectivity
- **Endpoint**: `https://www.apple.com/library/test/success.html`
- **Data Sent**: Standard HTTP request headers only
- **Data Received**: Simple success/failure response
- **Data Stored**: None

### 2. Speed Test
- **Purpose**: Measure download speed
- **Endpoint**: `https://httpbin.org/bytes/512000`
- **Data Sent**: Standard HTTP request headers only
- **Data Received**: 512KB test file
- **Data Stored**: Speed measurement result (locally only)

**No identifying information** is included in these requests beyond standard HTTP headers sent by iOS.

## Data Security

Since we don't collect any data, there is no data to secure on our end. Your local history data is secured by iOS's standard data protection mechanisms.

## Data Deletion

To delete your locally stored connection history:
1. Open the App
2. Tap "View History"
3. Tap "Clear" in the top right
4. Confirm deletion

Alternatively, deleting the App will remove all locally stored data.
